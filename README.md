# SmileCare - Premium Dental Clinic Website

A world-class dental clinic website built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## ✨ Features

- ✅ 14 Premium Sections
- ✅ 7 Reusable Components
- ✅ Framer Motion Animations
- ✅ Dark/Light Mode
- ✅ Mobile Responsive
- ✅ SEO Optimized
- ✅ Accessible (WCAG 2.1 AA)
- ✅ Lighthouse 95+

## 📦 Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Hook Form
- React Router
